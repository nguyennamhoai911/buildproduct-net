# BuildProduct.net

## Project Structure
- **frontend-user**: Next.js 15 Application (User facing Hub).
- **frontend-admin**: React + Vite Application (Admin Dashboard).
- **backend**: Node.js + Express API.

## Setup

### Backend (High Performance)
```bash
# Requires Bun (https://bun.sh)
cd backend
bun install
bun run dev
# Runs on port 3040
```

### Frontend User
```bash
cd frontend-user
npm install
npm run dev
# Runs on port 3041
```

### Frontend Admin
```bash
cd frontend-admin
npm install
npm run dev
# Runs on port 3032
```
